【*】程序简介

-工程名称：MPU-6050使用串口显示DMP解算姿态数据
-实验平台: 野火小智 STM32 F103C8T6 核心板

驱动MPU-6050，调用MPL软件库(Motion Processing Library)，解算姿态数据


/*********************************************************************************************/
【*】实验操作与注意事项：

实验操作
1.核心板掉电情况下，将模块引出脚按照xxx引脚连接说明.xlsx，接到板子对应脚上
2.确保引脚对应接上的情况下，核心板上电，将例程编译成功后，下载到核心板里
3.核心板的USB转串口通过数据线连接电脑，同时确保电脑安装了串口驱动并能识别到核心板的串口
4.电脑端使用串口调试助手，选择电脑与核心板相连的COM口，设置为115200-N-8-1并打开
5.复位核心板，即可接收核心板串口发送给电脑的数据
6.查看串口调试助手打印的提示信息

注意事项
MPU6050的AD0引脚默认接GND，地址为0x68 ,当接3V3时，地址改为0x69，可在mpu6050.h文件修改宏MPU6050_SLAVE_ADDRESS的值来匹配硬件连接
默认使用软件I2C，可在mpu6050.h文件中注释掉#define  soft_IIC使用硬件I2C

/*********************************************************************************************/
【*】引脚分配

MPU-6050：

		3V3 <--->3V3
		GND	<--->GND
		SDA	<--->PB11
		SCL	<--->PB10
		INT <--->PA11
		AD0 <--->GND

LED：
        LED1<--->PA1
        LED2<--->PA2
        LED3<--->PA3
		
串口（TTL-USB TO UART）：
CH340的收发引脚与STM32的发收引脚相连
        CH340  RXD  <--->  USART1  TX  (PA9)
        CH340  TXD  <--->  USART1  RX  (PA10)
